ADVANCED COMMANDS

GREP

GREP Command in Unix:
➤ Used to determine whether the search criteria are available in the file, and if they are, it prints the matching lines

Whoami

Whoami Command in Unix:
➤ Used to identify who is logged into the server (username)

Passwd

Passwd Command in Unix:
➤ Used to change the password
➤ It will ask for the old password and then prompt you to choose a new one

Kill

Kill Command in Unix:
➤ Used to stop an already executing process
➤ It is necessary to locate the PID of the process that should be killed and use that PID as the argument

Ps-Ef

Ps -Ef Command in Unix:
➤ Used to find the PID (Process ID, Unique number of the process) of the process

ftp

ftp Command in Unix:
➤ Used to transfer the files from one server to another server. All you need is the server's name and password

Find

Find Command in Unix:
➤ Used to find the files from the specified directory

Date

Date Command in Unix:
➤ Used to learn the current time in a server
    • ➣ current month – date + %m
    • ➣ current day of month – date + %d
    • ➣ the year – date + %y
    • ➣ the hour – date + %H
    • ➣ the minute – date + %M
    • ➣ the second – date + %S

Clear

Clear Command in Unix:
➤ Used to clear the screen
➤ The current window's content is cleaned, and the cursor is moved to the first line of the window

Sort

Sort Command in Unix:
➤ Used to arrange the lines of text alphabetically
    • ➣ SORT -N: - sorts text into alphabets and numeric lines sorts from low to high value
    • ➣ SORT -R: - sorts text from the files in reverse order
    • ➣ SORT -F: - sorts upper and lowercase texts together

Aliases

Aliases Command in Unix:
➤ Used to create a representation for another command
➤ The command clear could be written as "c" after using alias c clear command

Gzip

Gzip Command in Unix:
➤ Used to compress files
    • ➣ gzip filename

Gunzip

Gunzip Command in Unix:
➤ Used for uncompressing files compressed by gzip
    • ➣ gunzip filename

Gzcat

GGzcat Command in Unix:
➤ Used for looking at a zipped file without actually having to gunzip it
    • ➣ gzcat filename
