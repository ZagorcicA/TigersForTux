BASIC COMMANDS

List content

ls

  • ➤ Shows files/directories in current directory
  • ➤ Directories – in blue
  • ➤ Files – in white
    • ➣ ls -R (all files not only in directories but it also shows subdirectories)
    • ➣ ls -al (shows detailed information of the files)
    • ➣ ls -a (shows hidden files)

Display, copy & create text files

cat

  1. cat > filename (creates text file)
  2. Write the content of the text file
  3. Ctrl + d to exit
  4. cat filename - to see the file
    • ➣ cat file file2 > newfilename (combines 2 text files, cat newfilename to see the text file)

This command only works for text files.

Remove file

rm

  • ➤ Remove a file:
    • ➣ rm filename

Move file

mv

  • ➤ Move a file:
    • ➣ mv filename new_file_location
  • ➤ Renaming the file:
    • ➣ mv filename newname

Sudo command

sudo

  • ➤ Allows the user to run programs with the security privileges of another user
  • ➣ sudo your_command

Make directory

mkdir

  • ➤ Create a directory:
    • ➣ mkdir directoryname (or mkdir new_directory_location)

Remove directory

rmdir

  • ➤ Remove a directory:
    • ➣ rmdir directoryname newdirectoryname

Open manual

man

  • ➤ Reference book of a Linux operating system:
    • ➣ man mkdir (shows manual for mkdir command)

Show history

history

  • ➤ Displays all past commands used
    • ➣ a built-in keyword of the shell - variations depending on which one is being used

Clear terminal

clear

  • ➤ Clears the terminal
    • ➣ the resulting display will depend on the terminal that is being used

Formatting

pr

  • ➤ Used for formatting the files for printing on the terminal
    • ➣ pr -x filename (shows content divided in x columns)
    • ➣ pr -h “Header“ filename (header value is assigned as the header)
    • ➣ pr -n filename (denotes all the lines in the file with numbers)

Print files

lp, lpr

  • ➤ Used for printing files
    • ➣ lp filename (lpr filename)
  • ➤ Multiple copies:
    • ➣ 10 : lp -n10 filename (lp 10 testfile)

Edit packages

apt

  • ➤ Installing and removing packages:
    • ➣ sudo apt - get update (update all installed packages)
    • ➣ sudo apt - get install packagename (install packages)
